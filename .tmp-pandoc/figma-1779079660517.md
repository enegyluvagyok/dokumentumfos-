A Figma integráció lehetővé teszi, hogy bármely journey lépéshez csatolj egy Figma keretlinket. Amikor az integráció engedélyezve van, egy **Figma** gomb jelenik meg inline minden lépésen, azonnali hozzáférést biztosítva a tesztelőknek az éppen validált tervhez anélkül, hogy manuálisan füleket kellene váltaniuk.

## Hogyan működik

Az Analyse egyetlen globális Figma API tokent tárol. Amikor a token jelen van és az **Figma gombok engedélyezése** kapcsoló be van kapcsolva, minden olyan journey lépésnél, amelyhez Figma link van tárolva, megjelenik egy kattintható gomb, amely megnyitja a csatolt keretet a Figmában.

> **Információ:** A Figma integráció egyszer kerül konfigurálásra az alkalmazás szintjén, és az összes projektre vonatkozik.

## Az integráció konfigurálása

  
### A Figma konfiguráció megnyitása

    Az oldalsávban navigálj a **Beállítások → Figma konfiguráció** menüpontra.
  

  
### Add meg a Figma API tokent

    Illeszd be a személyes Figma hozzáférési tokenedet a **Figma API token** mezőbe. A mező takart — ha már van token
    mentve, egy pontokból álló helykitöltőt fogsz látni (`••••••••••••••••••••••••`). Csak akkor írd felül, ha a meglévő
    tokent szeretnéd lecserélni.
  

  
### Gombok engedélyezése

    Kapcsold be a **Figma gombok engedélyezése** kapcsolót. Ez szabályozza, hogy a Figma gomb globálisan megjelenjen-e a
    journey lépéseken. Bármikor kikapcsolhatod az összes gomb elrejtéséhez anélkül, hogy eltávolítanád a tokent.
  

  
### Mentés
Kattints a **Mentés** gombra. Az új beállítások azonnal érvénybe lépnek.

> **Figyelmeztetés:** 
  A token titkosítva tárolódik az adatbázisban, és mentés után sosem jelenik meg egyszerű szövegként. Ha elveszíted a
  tokent, egy újat kell generálnod a Figmában, és újra be kell illesztened.

## Figma hozzáférési token generálása

Személyes hozzáférési token létrehozásához a Figmában:

  
### A Figma beállítások megnyitása

    A Figmában kattints az avatarodra a bal felső sarokban, majd menj a **Beállítások** menüpontra.
  

  
### Görgess a Személyes hozzáférési tokenekhez

    A **Biztonság** szakasz alatt keresd meg a **Személyes hozzáférési tokenek** részt, és kattints az **Új token
    generálása** gombra.
  

  
### Másold le és mentsd el a tokent

    Azonnal másold le a tokent — a Figma csak egyszer mutatja meg. Illeszd be az Analyse **Figma API token** mezőjébe.
  

> **Tipp:** 
  A tokennek legalább 20 karakter hosszúnak kell lennie. A Figma személyes hozzáférési tokenek általában sokkal
  hosszabbak, így egy érvényes token mindig megfelel ennek a követelménynek.

## Figma linkek csatolása lépésekhez

Az integráció konfigurálása után bármely journey lépéshez tárolhatsz Figma linket:

1. Nyiss meg egy journey verziót, és szerkessz vagy hozz létre egy lépést.
2. Add meg a teljes Figma keret URL-jét a **Figma** mezőben (pl. `https://www.figma.com/file/…`).
3. Mentsd el a lépést.

Amikor a **Figma gombok engedélyezése** be van kapcsolva, és egy lépéshez van Figma link csatolva, a tesztvégrehajtás során megjelenik egy gomb az adott lépésen. Kattintásra megnyílik a csatolt keret a Figmában.

> **Megjegyzés:** 
  A Figma link nélküli lépéseknél nem jelenik meg a gomb, még akkor sem, ha az integráció engedélyezett. A linkek
  tömegesen is importálhatók a lépések Excel-sablonjával — részletekért lásd az [Export és
  import](/integrations/export-import) oldalt.
